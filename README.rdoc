= ruby-batch

http://github.com/bjornno/ruby-batch/

== DESCRIPTION:
Simple DSL for batch processing

== FEATURES/PROBLEMS:
Not very useful yet
Not multithreaded

== SYNOPSIS:

# Scans for files matching pattern in catalog

filescan '/data/from_host/', 'processfiles*' do |file|   
  # process file logic
end

# Scans for work in database

dbscan db, 'process_queue' do |work_item|   
  # process work_item logic
end

* FIX (list of requirements)

== INSTALL:

sudo gem install bjornno-ruby-batch
